# Калькулятор ИМТ
Приложение сфокусированно на своей основной функции - расчете ИМТ.

## Основные возможности
Основная возможность - быстрый расчет ИМТ с приятным интерфейсом.\

Реализована функция ввода данных пользователя (вес и рост в метрической системе) и кнопка "Очистить", если на нее нажать, то данные будут очищены.
